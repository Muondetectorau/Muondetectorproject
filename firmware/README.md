##  О прошивке

Данный код превращает **ESP32** в автономный регистратор частиц. Из-за высокой скорости сигналов от SiPM (длительность импульсов < 1 мкс), прошивка использует **многопоточность (FreeRTOS)** для разделения критичных ко времени задач и задач обслуживания сети.

Устройство создает собственную Wi-Fi точку доступа, предоставляя удобный веб-интерфейс для мониторинга попаданий (мюонов), настройки чувствительности "на лету" и скачивания логов.

---

##  Ключевые возможности

-  **Двухъядерная обработка:** 
  - **Ядро 0:** Выделено исключительно под непрерывный высокоскоростной опрос АЦП (ADC polling). Сторожевой таймер (WDT) отключен для предотвращения прерываний.
  - **Ядро 1:** Обслуживает Wi-Fi, Web-сервер и запись данных во flash-память.
-  **Встроенная точка доступа (SoftAP):** Не требует внешнего роутера. Работает "в поле".
-  **Логирование в LittleFS:** Данные сохраняются в энергонезависимую память в формате `.csv` (Время, Амплитуда пика).
-  **Адаптивный Web-интерфейс:**  Gанель управления, оптимизированная для смартфонов и ПК.

##  Подключение (Pinout)

Прошивка настроена на использование следующих выводов ESP32:

| Пин | Назначение | Описание |
| :---: | :--- | :--- |
| **`34`** | `PIN_SIGNAL` | Вход АЦП (Analog Read). Подключается к выходу усилителя / Fast Output SiPM. |
| **`2`** | `PIN_LED` | Индикаторный светодиод (обычно встроенный на плате ESP32). Вспыхивает при регистрации частицы. |

> **Важно:** Для АЦП установлено затухание `ADC_0db`. Это обеспечивает максимальную чувствительность, но ограничивает измеряемый диапазон напряжением **~1.1В**. Убедитесь, что сигнал с сенсора не превышает это значение.

##  Установка и запуск

1. Откройте проект в **Arduino IDE** или **PlatformIO**.
2. Убедитесь, что выбрана плата на базе ESP32 (например, *DOIT ESP32 DEVKIT V1*).
3. Скомпилируйте и прошейте контроллер.
4. Включите питание устройства. 
5. Подождите 1-2 секунды (в это время происходит расчет базового уровня шума).

##  Использование Web-интерфейса

После запуска ESP32 создаст Wi-Fi сеть:
*   **Имя сети (SSID):** `MuonDetector`
*   **Пароль:** `12345678`

Подключитесь к сети с телефона или ноутбука и откройте в браузере адрес:
 **`http://192.168.4.1`**

### Функции интерфейса:
*    **Счетчик мюонов:** Показывает общее количество зарегистрированных событий.
*   🎚 **Ползунок порога:** Позволяет отстроить детектор от фонового шума. Значение прибавляется к базовой линии (baseline). a Нажмите *«ПРИМЕНИТЬ ПОРОГ»* для сохранения.
*   ⬇ **СКАЧАТЬ CSV:** Скачивает файл `muon_log.csv` с историей всех срабатываний для последующего анализа (например, в Excel или Python).
*   🗑 **ОЧИСТИТЬ ЛОГ:** Удаляет старые данные и сбрасывает счетчик (требует подтверждения).
