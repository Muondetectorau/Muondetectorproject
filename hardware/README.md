##  Краткое описание архитектуры

Главная особенность данной схемы — **аппаратно-программный детектор пиков (Software Peak Detector)**. 
В отличие от классических схем, требующих дорогостоящих высокоскоростных компараторов, здесь применяется метод RC-растягивания импульса. Это позволяет оцифровывать наносекундные сигналы от квантового датчика (SiPM) с помощью стандартного встроенного АЦП микроконтроллера ESP32.

> **Ключевой принцип:** Токовый импульс от мюона преобразуется в напряжение и растягивается с 50 наносекунд до ~100 микросекунд с помощью нагрузочного резистора на 47 кОм.

## Логические блоки схемы

Схема разделена на 4 функциональных блока для минимизации радиочастотных помех:

1. **Блок Питания (HV Boost):** Преобразование 5V от USB (PowerBank) в 29V для питания SiPM с помощью модуля XL6009.
2. **Блок Фильтрации (RC-Filter):** Подавление импульсных помех и высокочастотного шума питания перед подачей на катод SiPM ($R=1\text{кОм}$, $C=10\text{мкФ} \parallel 100\text{нФ}$).
3. **Формирователь импульса (Pulse Shaper):** Нагрузочная цепь на аноде SiPM ($R=47\text{кОм}$) и разделительный конденсатор ($100\text{нФ}$) для отсечения постоянного темнового тока.
4. **Блок Оцифровки (MCU):** Согласование сигнала с аналоговым входом ESP32 (пин `IO34`), стянутым к земле ($R=10\text{кОм}$).

## Спецификация компонентов (BOM)

| Обозначение | Компонент | Номинал / Тип | Назначение |
| :---: | :--- | :--- | :--- |
| **U1** | Микроконтроллер | ESP32-WROOM-32 (DevKit) | Оцифровка, логика, запись данных |
| **U2** | Квантовый датчик | MicroFC-60035-SMT-TR | Регистрация фотонов (SiPM) |
| **U3** | Модуль питания | XL6009 /  | Генерация смещения 29V |
| **R1** | Резистор | 1 кОм (0.25 Вт) | Фильтр питания катода |
| **R2** | Резистор | 40 кОм (0.25 Вт) | Растягивание импульса (Anode) |
| **R3** | Резистор | 10 кОм (0.25 Вт) | Стяжка аналогового пина  |
| **C1** | Конденсатор (Электролит) | 10 мкФ (≥50V) | Низкочастотный фильтр питания |
| **C2, C3**| Конденсатор (Керамика) | 100 нФ / 104 (50V) | ВЧ-фильтр и гальваническая развязка |
