##  О проекте

Данный проект , использует низкошумящие кремниевые фотоэлектронные умножители (SiPM) **C-Series** производства **onsemi**. 

Сенсоры обладают характеристиками, аналогичными традиционным фотоэлектронным умножителям (ФЭУ - PMT), но предлагают все практические преимущества твердотельной технологии: низкое рабочее напряжение, превосходную температурную стабильность, компактность и низкую стоимость.



##  Ключевые особенности (Hardware)

-  **Чувствительность:** Оптимизированы для синего спектра (пик на 420 нм). Широкий диапазон 300–950 нм.
-  **Низкий уровень шума:** Индустриальный стандарт низкого уровня темнового счета (Dark Count Rate) в сочетании с высокой эффективностью детектирования фотонов (PDE).
-  **Удобный форм-фактор:** SMT корпуса (1 мм, 3 мм, 6 мм) совместимые со стандартной пайкой оплавлением (Lead-free).

| Параметр | Значение (Тип.) | Ед. изм. | Примечание |
| :--- | :---: | :---: | :--- |

<details>
<summary><b> Развернуть подробности о конфигурации выводов</b></summary>

Сенсор представляет собой массив микроячеек, суммированных вместе, и работает как единый фотодиод с 3 выводами:
1. **Anode** (Анод) — Стандартный выход
2. **Cathode** (Катод) — Подача смещения (Vbias)
3. **Fast Output** — Быстрый выход для прецизионного тайминга
