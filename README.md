# Мюонный детектор на SiPM и сцинтилляторе

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Hardware: Open Source](https://img.shields.io/badge/Hardware-Open_Source-green.svg)](#)

Этот проект представляет собой открытый  аппаратно-программный комплекс для регистрации космических лучей (мюонов). Детектор построен на базе пластикового сцинтиллятора и кремниевого фотоумножителя (SiPM) серии **MicroFC** (SensL / ON Semiconductor).

##  Описание проекта

Космические лучи (в основном протоны) постоянно бомбардируют атмосферу Земли, создавая каскады вторичных частиц. До поверхности земли долетают преимущественно **мюоны**. 

Данный детектор фиксирует слабые вспышки света (фотонные лавины), возникающие при прохождении высокоэнергетичного мюона через пластиковый сцинтиллятор. Эти вспышки регистрируются сверхчувствительным твердотельным сенсором — SiPM (MicroFC). Сигнал с сенсора усиливается, формируется и считывается микроконтроллером для дальнейшего анализа.

В отличие от классических детекторов на вакуумных ФЭУ (фотоэлектронных умножителях), этот проект портативен и **не требует высоких напряжений** (для работы SiPM нужно около 30 В).

##  Ключевые особенности

* **Безопасность:** Низкое рабочее напряжение смещения (~25-30V) по сравнению с ФЭУ (~1000V).
* **Компактность:** Использование твердотельного датчика MicroFC позволяет сделать детектор настольным или карманным.
* **Высокая чувствительность:** Способность регистрировать единичные фотоны.
* **Доступность:** Использование популярных микроконтроллеров (ESP32, STM32, Arduino) для подсчета совпадений или импульсов.
* **Мониторинг:** Логирование данных во внутреннее хранилище, или передача на ПК через Serial/Wi-Fi.

## 🛠 Компоненты 

### Электроника
1. **SiPM сенсор:** `MicroFC-60035-SMT`.
2. **Микроконтроллер:** ESP32 
3. **Источник напряжения смещения:** DC-DC повышающий преобразователь с низким уровнем шума (регулируемый от 20V до 35V для питания MicroFC).

### Оптика и механика
1. **Сцинтиллятор:** Пластиковый сцинтиллятор.
2. **Оптический контакт:** Оптическая смазка  или прозрачный силиконовый компаунд для соединения сцинтиллятора и окна MicroFC.
3. **Светоотражатель:** Тефлоновая лента (ФУМ) для обмотки сцинтиллятора.
4. **Светоизоляция:** Черная изолента, светонепроницаемый корпус.
5. **Внимание: SiPM мгновенно засвечивается (и может деградировать) от дневного света при поданном питании!**
### Физическая модель
Интерактивная физическая симуляция прохождения космических мюонов через вещество и калибровка сцинтилляционного детектора.

## Что внутри:
-  Честный Монте-Карло движок (Ray-marching) потерь энергии по формуле Бете-Блоха.
-  Моделирование спектра Ландау в тонких пластиковых сцинтилляторах.
-  Оптимизация соотношения Сигнал/Шум при наличии радиационного фона.
-  Анализ статистики Пуассона и барометрического эффекта.

## Как запустить:
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите веб-интерфейс: `streamlit run web_app.py`
